# ONLAIS: каталог направлений для онлайн-записи

Статус: `v3 (на согласование)`  
Дата обновления: `2026-04-19`

Назначение каталога:
- направления только для платформы записи по слотам времени;
- учет сценариев: услуги, специалисты, индивидуальные и групповые записи;
- использование в регистрации бизнеса и автоподстановке стартового контента.

## 1. Критерии включения направления

Направление включается, если для него характерно:
- запись на конкретное время (слот);
- выбор услуги и/или специалиста;
- регулярные повторяющиеся визиты;
- возможность онлайн-управления расписанием.

## 3. Что исключено из каталога

Исключены направления, где нет типового сценария записи «услуга + слот + специалист/группа», например:
- логистика/доставка;
- ремонт техники/домашний ремонт;
- государственные/НКО сервисы;
- страхование;
- универсальное «другое».

## 4. Режимы записи по направлению

Для всех направлений поддерживаются 2 режима (можно включать один или оба):
- `INDIVIDUAL`: индивидуальные услуги (специалист + слот)
- `GROUP`: групповые занятия/события (группа + слот + лимит мест)

## 5. Правила для UI регистрации

- Пользователь выбирает одно основное направление на старте.
- Дополнительные направления можно добавить в настройках после регистрации.
- В UI показывать только русский label, в БД сохранять `key`.
- По выбранному направлению подставлять дефолтные текстовые шаблоны в конструкторе сайта.
- Услуги, категории, длительности и расписание владелец бизнеса заполняет вручную в CRM.

## 6. Кандидаты на второй этап (не включены сейчас)

Оставлены на отдельное решение (если потребуется):
- аренда конференц-залов;
- медицинские телемосты/онлайн-консультации как отдельный режим;
- B2B-услуги с нестандартной длительностью и без жестких слотов.
